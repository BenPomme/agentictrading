# Support

Use the right channel so questions, bugs, and security reports do not get mixed together.

## Where To Ask What

- GitHub Issues: reproducible bugs, broken docs, and feature requests
- GitHub Discussions: setup questions, architecture discussion, and public design conversations
- Security vulnerabilities: follow [SECURITY.md](SECURITY.md), not public issues

## Before Opening An Issue

- read [README.md](README.md)
- check [docs/OPEN_SOURCE_BOUNDARY.md](docs/OPEN_SOURCE_BOUNDARY.md)
- search existing issues and discussions first

## What Maintainers Need

When asking for help, include:

- what you were trying to do
- what environment you are using
- commands run
- logs or stack traces
- whether the issue is specific to Meerkat, MobKit, Goldfish, or the AgenticTrading control plane

## Scope Reminder

This public repo supports paper-mode research and reference-app workflows. Questions that depend on private datasets, venue credentials, or live production trading infrastructure may be declined or answered only at a high level.
